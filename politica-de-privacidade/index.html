Política de Privacidade

Última atualização: março de 2026

1. Quem somos

O PesCam é um aplicativo Android para gravação de pescaria com buffer retroativo. Desenvolvido de forma independente por Felipe, Cuiabá-MT. Contato: admin@pescam.com.br

2. Dados coletados

O PesCam não coleta, armazena, transmite nem compartilha nenhum dado pessoal dos usuários em servidores externos. Tudo o que o app acessa permanece exclusivamente no seu dispositivo.

3. Permissões utilizadas

  • CAMERA: necessária para visualizar e gravar vídeos
  • RECORD_AUDIO: necessária para gravar áudio junto com o vídeo
  • WAKE_LOCK: mantém o processador ativo durante a gravação em segundo plano
  • FOREGROUND_SERVICE: permite que a gravação continue com a tela apagada
  • READ_MEDIA_VIDEO / READ_EXTERNAL_STORAGE: permite listar os vídeos salvos dentro do app

Nenhuma dessas permissões é usada para coletar dados ou rastrear o usuário.

4. Assinatura Premium

O processamento de pagamentos é realizado exclusivamente pelo Google Play. O PesCam não tem acesso a dados de cartão de crédito ou informações bancárias.

5. Dados de terceiros

O PesCam utiliza o Google Play Billing para gerenciar assinaturas, sujeito à Política de Privacidade do Google.

6. Crianças

O PesCam não é direcionado a crianças menores de 13 anos e não coleta intencionalmente dados de menores.

7. Alterações

Podemos atualizar esta política ocasionalmente. Quando isso ocorrer, atualizaremos a data no topo desta página.
